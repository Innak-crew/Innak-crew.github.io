API Documentation

Endpoints

Method Endpoint Description
POST /addStudent Add a new student.
GET /listStudents Retrieve a list of all students.
GET /getStudent/:id Retrieve a specific student by id.
PUT /updateStudent/:id Update an existing student.
DELETE /deleteStudent/:id Delete a specific student by id.
POST /addQuestion Add a new question.
GET /listQuestions Retrieve a list of all questions.
GET /getQuestion/:id Retrieve a specific question by id.
PUT /updateQuestion/:id Update an existing question.
DELETE /deleteQuestion/:id Delete a specific question by id.
POST /addStudentProgram Add a new Student Program.
GET /listStudentPrograms Retrieve a list of all Student Programs.
GET /getStudentprogram/:id Retrieve a specific Student Program by id.
